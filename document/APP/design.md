## アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | TriggerSearch |
| 略称 | trigs |
| 目的 | 「頭痛」「アレルギー」などの症状の原因（トリガー）を発見する |
| 設計方針 | お年寄りでも使える超シンプルなUI、複雑性は一切排除 |

---

## 機能一覧

### 毎日の記録（入力）

| 機能 | 説明 |
|------|------|
| 症状記録 | 「頭痛」「アレルギー」が起きたか（はい/いいえ） |
| 食事記録 | 朝・昼・夜に何を食べたか（選択式 or 簡単入力） |
| 睡眠記録 | 何時間寝たか（スライダーで選択） |
| 天気記録 | 晴れ/曇り/雨（自動取得 or 選択） |

### 分析（自動）

| 機能 | 説明 |
|------|------|
| トリガー分析 | 蓄積データから症状の原因を自動推測 |
| 結果表示 | 「〇〇を食べた日は頭痛が多い」などシンプルに表示 |
