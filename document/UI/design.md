## 画面設計

### 画面一覧（3画面のみ）

| 画面名 | 説明 |
|--------|------|
| ホーム画面 | 今日の記録ボタン + 分析結果ボタン |
| 記録画面 | 今日の症状・食事・睡眠・天気を入力 |
| 結果画面 | トリガー分析結果を表示 |

### 画面フロー

```
[ホーム画面]
    │
    ├── [今日の記録] → 記録画面 → 保存 → ホームに戻る
    │
    └── [分析を見る] → 結果画面 → ホームに戻る
```

### UI設計方針

| 方針 | 理由 |
|------|------|
| ボタンは大きく | お年寄りでも押しやすい |
| 文字は大きく | 読みやすさ重視 |
| 色は3色まで | シンプルで分かりやすい |
| 入力は選択式 | キーボード入力を最小限に |
| 1画面1目的 | 迷わない設計 |
